## Role
You are an Infrastructure Automation Engineer who deploys N8N on [your-railway] with PostgreSQL —
entirely through GitHub Actions and the [your-railway] GraphQL API — no dashboards, no manual CLI.
All credentials are fetched keylessly from GCP Secret Manager via Workload Identity Federation.

### Step 2: WIF — How Keyless Auth Works

WIF is what makes the entire pipeline autonomous. No stored keys, no rotation risk.

```
GitHub Actions job starts
  → GitHub issues a short-lived OIDC token (proves: repo=this repo, TTL=1 job)
  → GCP WIF verifies the token against allowlist (edri2or/* repos only)
  → GCP issues short-lived access credential (TTL=60 min)
  → Workflow reads secrets from GCP Secret Manager
  → Credential expires — nothing to clean up
```

No action needed here — just confirm the GitHub Secrets from Step 1 are set.

### Step 3: Write .github/workflows/deploy-n8n.yml

If the file already exists, print:
`deploy-n8n.yml already exists. Overwrite? (y/n)` — wait for confirmation.

Write the file below. Replace `<project-name>` with the name from `git remote get-url origin`:

```yaml
name: Deploy N8N + PostgreSQL on Railway

on:
  workflow_dispatch:

permissions:
  id-token: write
  contents: read

env:
  N8N_DOMAIN: n8n.life130.or-infra.com
  N8N_ADMIN_EMAIL: admin@life130.or-infra.com
  PG_IMAGE: ghcr.io/railwayapp-templates/postgres-ssl:17
  N8N_IMAGE: n8nio/n8n:latest
  RAILWAY_GQL: https://backboard.railway.com/graphql/v2

jobs:
  deploy:
    name: Bootstrap Railway + Deploy N8N
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Authenticate to GCP via WIF
        uses: google-github-actions/auth@v2
        with:
          workload_identity_provider: ${{ secrets.WIF_PROVIDER }}
          service_account: ${{ secrets.WIF_SERVICE_ACCOUNT }}

      - name: Set up gcloud CLI
        uses: google-github-actions/setup-gcloud@v2

      - name: Generate N8N secrets if not present
        env:
          GCP_PROJECT_ID: ${{ secrets.GCP_PROJECT_ID }}
        run: |
          generate_secret() {
            local name="$1"
            VERSIONS=$(gcloud secrets versions list "$name" \
              --project="$GCP_PROJECT_ID" --format=json 2>/dev/null | jq 'length')
            if [ "$VERSIONS" -gt 0 ]; then
              echo "SKIP $name"
            else
              openssl rand -hex 32 | tr -d '\n' | \
                gcloud secrets versions add "$name" \
                  --project="$GCP_PROJECT_ID" --data-file=-
              echo "OK $name"
            fi
          }
          generate_secret "N8N_ENCRYPTION_KEY" &
          generate_secret "N8N_BASIC_AUTH_PASSWORD" &
          generate_secret "N8N_OWNER_PASSWORD" &
          wait

      - name: Fetch secrets from GCP Secret Manager
        id: secrets
        uses: google-github-actions/get-secretmanager-secrets@v2
        with:
          secrets: |-
            RAILWAY_TOKEN:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/RAILWAY_TOKEN/versions/latest
            CLOUDFLARE_TOKEN:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/CLOUDFLARE_TOKEN/versions/latest
            CLOUDFLARE_ZONE_ID:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/CLOUDFLARE_ZONE_ID/versions/latest
            N8N_ENCRYPTION_KEY:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/N8N_ENCRYPTION_KEY/versions/latest
            N8N_BASIC_AUTH_PASSWORD:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/N8N_BASIC_AUTH_PASSWORD/versions/latest
            N8N_OWNER_PASSWORD:projects/${{ secrets.GCP_PROJECT_ID }}/secrets/N8N_OWNER_PASSWORD/versions/latest

      - name: Create Railway project (idempotent)
        id: railway
        env:
          RAILWAY_TOKEN: ${{ steps.secrets.outputs.RAILWAY_TOKEN }}
          KNOWN_PROJECT_ID: ${{ secrets.RAILWAY_PROJECT_ID }}
        run: |
          if [ -n "$KNOWN_PROJECT_ID" ]; then
            PROJ=$(curl -sf -X POST "$RAILWAY_GQL" \
              -H "Authorization: Bearer $RAILWAY_TOKEN" \
              -H "Content-Type: application/json" \
              -d "$(jq -cn --arg id "$KNOWN_PROJECT_ID" \
                '{"query":"query($id:String!){project(id:$id){id name environments{edges{node{id name}}}}}","variables":{"id":$id}}')")
            read -r PROJECT_ID ENV_ID < <(echo "$PROJ" | jq -r \
              '[(.data.project.id // ""), (.data.project.environments.edges[0].node.id // "")] | @tsv')
            if [ -n "$PROJECT_ID" ]; then
              echo "Project already exists — reusing."
              echo "project_id=$PROJECT_ID" >> "$GITHUB_OUTPUT"
              echo "env_id=$ENV_ID" >> "$GITHUB_OUTPUT"
              exit 0
            fi
            echo "::warning::Stored project ID not found — scanning workspace"
          fi

          WS_RESP=$(curl -sf -X POST "$RAILWAY_GQL" \
            -H "Authorization: Bearer $RAILWAY_TOKEN" \
            -H "Content-Type: application/json" \
            -d '{"query":"{ me { workspaces { id name projects { edges { node { id name environments { edges { node { id name } } } } } } } } }"}')
          WS_DATA=$(echo "$WS_RESP" | jq -c '.data.me.workspaces[0]')
          WORKSPACE_ID=$(echo "$WS_DATA" | jq -r '.id // empty')
          if [ -z "$WORKSPACE_ID" ]; then
            echo "::error::Could not resolve Railway workspace ID"
            exit 1
          fi

          PROJECT_NODE=$(echo "$WS_DATA" | jq -c \
            '[.projects.edges[].node | select(.name == "<project-name>")] | first')
          PROJECT_ID=$(echo "$PROJECT_NODE" | jq -r '.id // empty')

          if [ -n "$PROJECT_ID" ]; then
            echo "Project already exists — reusing."
            ENV_ID=$(echo "$PROJECT_NODE" | jq -r '.environments.edges[0].node.id // empty')
          else
            echo "Creating Railway project..."
            PAYLOAD=$(jq -cn --arg wsId "$WORKSPACE_ID" \
              '{"query":"mutation($wsId:String!){projectCreate(input:{name:\"<project-name>\",workspaceId:$wsId,defaultEnvironmentName:\"production\"}){id environments{edges{node{id name}}}}}","variables":{"wsId":$wsId}}')
            RESPONSE=$(curl -sf -X POST "$RAILWAY_GQL" \
              -H "Authorization: Bearer $RAILWAY_TOKEN" \
              -H "Content-Type: application/json" \
              -d "$PAYLOAD")
            PROJECT_ID=$(echo "$RESPONSE" | jq -r '.data.projectCreate.id')
            ENV_ID=$(echo "$RESPONSE" | jq -r '.data.projectCreate.environments.edges[0].node.id')
          fi

          if [ -z "$PROJECT_ID" ] || [ "$PROJECT_ID" = "null" ]; then
            echo "::error::Failed to create or find Railway project"
            exit 1
          fi

          echo "project_id=$PROJECT_ID" >> "$GITHUB_OUTPUT"
          echo "env_id=$ENV_ID" >> "$GITHUB_OUTPUT"

      - name: Create Railway services — PostgreSQL + N8N (idempotent)
        id: services
        env:
          RAILWAY_TOKEN: ${{ steps.secrets.outputs.RAILWAY_TOKEN }}
          PROJECT_ID: ${{ steps.railway.outputs.project_id }}
        run: |
          ALL_SERVICES=$(curl -sf -X POST "$RAILWAY_GQL" \
            -H "Authorization: Bearer $RAILWAY_TOKEN" \
            -H "Content-Type: application/json" \
            -d "$(jq -cn --arg pid "$PROJECT_ID" \
              '{"query":"query($pid:String!){project(id:$pid){services{edges{node{id name}}}}}","variables":{"pid":$pid}}')")

          create_service() {
            local name="$1" image="$2" out_var="$3"
            local SERVICE_ID
            SERVICE_ID=$(echo "$ALL_SERVICES" | jq -r \
              --arg n "$name" \
              '[.data.project.services.edges[].node | select(.name == $n)] | first | .id // empty')

            if [ -n "$SERVICE_ID" ]; then
              echo "$name already exists — reusing: $SERVICE_ID"
            else
              echo "Creating $name service..."
              local PAYLOAD
              PAYLOAD=$(jq -cn --arg pid "$PROJECT_ID" --arg n "$name" --arg img "$image" \
                '{"query":"mutation($pid:String!,$n:String!,$img:String!){serviceCreate(input:{projectId:$pid,name:$n,source:{image:$img}}){id name}}","variables":{"pid":$pid,"n":$n,"img":$img}}')
              local RESPONSE
              RESPONSE=$(curl -sf -X POST "$RAILWAY_GQL" \
                -H "Authorization: Bearer $RAILWAY_TOKEN" \
                -H "Content-Type: application/json" \
                -d "$PAYLOAD")
              SERVICE_ID=$(echo "$RESPONSE" | jq -r '.data.serviceCreate.id')
            fi

            if [ -z "$SERVICE_ID" ] || [ "$SERVICE_ID" = "null" ]; then
              echo "::error::Failed to create $name service"
              exit 1
            fi
            echo "$out_var=$SERVICE_ID" >> "$GITHUB_OUTPUT"
          }

          create_service "Postgres" "$PG_IMAGE" "pg_service_id"
          create_service "n8n" "$N8N_IMAGE" "n8n_service_id"

      - name: Configure N8N environment variables
        env:
          RAILWAY_TOKEN: ${{ steps.secrets.outputs.RAILWAY_TOKEN }}
          PROJECT_ID: ${{ steps.railway.outputs.project_id }}
          ENV_ID: ${{ steps.railway.outputs.env_id }}
          N8N_SERVICE_ID: ${{ steps.services.outputs.n8n_service_id }}
          N8N_ENCRYPTION_KEY: ${{ steps.secrets.outputs.N8N_ENCRYPTION_KEY }}
          N8N_BASIC_AUTH_PASSWORD: ${{ steps.secrets.outputs.N8N_BASIC_AUTH_PASSWORD }}
        run: |
          set_var() {
            local name="$1" value="$2"
            curl -sf -X POST "$RAILWAY_GQL" \
              -H "Authorization: Bearer $RAILWAY_TOKEN" \
              -H "Content-Type: application/json" \
              -d "$(jq -cn \
                --arg pid "$PROJECT_ID" --arg eid "$ENV_ID" \
                --arg sid "$N8N_SERVICE_ID" --arg n "$name" --arg v "$value" \
                '{"query":"mutation($pid:String!,$eid:String!,$sid:String!,$n:String!,$v:String!){variableUpsert(input:{projectId:$pid,environmentId:$eid,serviceId:$sid,name:$n,value:$v})}","variables":{"pid":$pid,"eid":$eid,"sid":$sid,"n":$n,"v":$v}}')" \
              | jq -r --arg n "$name" 'if .errors then "ERROR \(.errors[0].message)" else "OK \($n)" end'
          }
          set_var "N8N_HOST"                "$N8N_DOMAIN" &
          set_var "N8N_PROTOCOL"            "https" &
          set_var "N8N_PORT"                "5678" &
          set_var "WEBHOOK_URL"             "https://$N8N_DOMAIN/" &
          set_var "N8N_ENCRYPTION_KEY"      "$N8N_ENCRYPTION_KEY" &
          set_var "N8N_BASIC_AUTH_ACTIVE"   "true" &
          set_var "N8N_BASIC_AUTH_USER"     "admin" &
          set_var "N8N_BASIC_AUTH_PASSWORD" "$N8N_BASIC_AUTH_PASSWORD" &
          set_var "DB_TYPE"                 "postgresdb" &
          set_var "DB_POSTGRESDB_HOST"      '${{Postgres.PGHOST}}' &
          set_var "DB_POSTGRESDB_PORT"      '${{Postgres.PGPORT}}' &
          set_var "DB_POSTGRESDB_DATABASE"  '${{Postgres.PGDATABASE}}' &
          set_var "DB_POSTGRESDB_USER"      '${{Postgres.PGUSER}}' &
          set_var "DB_POSTGRESDB_PASSWORD"  '${{Postgres.PGPASSWORD}}' &
          set_var "DB_POSTGRESDB_SSL_CA"    '${{Postgres.SSL_CERT_PATH}}' &
          set_var "N8N_RUNNERS_ENABLED"     "true" &
          set_var "EXECUTIONS_MODE"         "regular" &
          wait

      - name: Register custom domain on N8N service
        env:
          RAILWAY_TOKEN: ${{ steps.secrets.outputs.RAILWAY_TOKEN }}
          ENV_ID: ${{ steps.railway.outputs.env_id }}
          N8N_SERVICE_ID: ${{ steps.services.outputs.n8n_service_id }}
        run: |
          PAYLOAD=$(jq -cn --arg eid "$ENV_ID" --arg sid "$N8N_SERVICE_ID" --arg d "$N8N_DOMAIN" \
            '{"query":"mutation($eid:String!,$sid:String!,$d:String!){customDomainCreate(input:{environmentId:$eid,serviceId:$sid,domain:$d}){id domain status}}","variables":{"eid":$eid,"sid":$sid,"d":$d}}')
          RESULT=$(curl -sf -X POST "$RAILWAY_GQL" \
            -H "Authorization: Bearer $RAILWAY_TOKEN" \
            -H "Content-Type: application/json" \
            -d "$PAYLOAD")
          echo "$RESULT" | jq '{domain: .data.customDomainCreate.domain, status: .data.customDomainCreate.status, errors: .errors}'

      - name: Create Cloudflare CNAME (idempotent)
        env:
          CLOUDFLARE_TOKEN: ${{ steps.secrets.outputs.CLOUDFLARE_TOKEN }}
          CLOUDFLARE_ZONE_ID: ${{ steps.secrets.outputs.CLOUDFLARE_ZONE_ID }}
          RAILWAY_TOKEN: ${{ steps.secrets.outputs.RAILWAY_TOKEN }}
          N8N_SERVICE_ID: ${{ steps.services.outputs.n8n_service_id }}
          ENV_ID: ${{ steps.railway.outputs.env_id }}
        run: |
          DOMAINS=$(curl -sf -X POST "$RAILWAY_GQL" \
            -H "Authorization: Bearer $RAILWAY_TOKEN" \
            -H "Content-Type: application/json" \
            -d "$(jq -cn --arg sid "$N8N_SERVICE_ID" --arg eid "$ENV_ID" \
              '{"query":"query($sid:String!,$eid:String!){serviceInstance(serviceId:$sid,environmentId:$eid){domains{serviceDomains{domain}}}}","variables":{"sid":$sid,"eid":$eid}}')")
          RAILWAY_DOMAIN=$(echo "$DOMAINS" | jq -r \
            '.data.serviceInstance.domains.serviceDomains[0].domain // empty')
          if [ -z "$RAILWAY_DOMAIN" ]; then
            echo "::warning::Railway service domain not yet assigned — using railway.app placeholder"
            RAILWAY_DOMAIN="railway.app"
          fi

          EXISTING=$(curl -sf \
            "https://api.cloudflare.com/client/v4/zones/$CLOUDFLARE_ZONE_ID/dns_records?name=$N8N_DOMAIN&type=CNAME" \
            -H "Authorization: Bearer $CLOUDFLARE_TOKEN")
          RECORD_ID=$(echo "$EXISTING" | jq -r '.result[0].id // empty')

          if [ -n "$RECORD_ID" ]; then
            curl -sf -X PATCH \
              "https://api.cloudflare.com/client/v4/zones/$CLOUDFLARE_ZONE_ID/dns_records/$RECORD_ID" \
              -H "Authorization: Bearer $CLOUDFLARE_TOKEN" \
              -H "Content-Type: application/json" \
              -d "{\"content\":\"$RAILWAY_DOMAIN\"}" | jq '{success: .success, content: .result.content}'
          else
            curl -sf -X POST \
              "https://api.cloudflare.com/client/v4/zones/$CLOUDFLARE_ZONE_ID/dns_records" \
              -H "Authorization: Bearer $CLOUDFLARE_TOKEN" \
              -H "Content-Type: application/json" \
              -d "{\"type\":\"CNAME\",\"name\":\"n8n.life130\",\"content\":\"$RAILWAY_DOMAIN\",\"ttl\":1,\"proxied\":false}" \
              | jq '{success: .success, id: .result.id, name: .result.name}'
          fi

      - name: E2E — Wait for N8N healthcheck (up to 10 min)
        run: |
          for attempt in {1..20}; do
            HTTP=$(curl -s -o /dev/null -w "%{http_code}" --max-time 10 \
              "https://$N8N_DOMAIN/healthz/readiness" || echo "000")
            echo "Attempt $attempt: HTTP $HTTP"
            [ "$HTTP" = "200" ] && { echo "N8N is up ✅"; exit 0; }
            sleep 30
          done
          echo "::error::N8N did not respond with HTTP 200 after 10 minutes"
          exit 1

      - name: Create N8N owner account
        env:
          N8N_OWNER_PASSWORD: ${{ steps.secrets.outputs.N8N_OWNER_PASSWORD }}
        run: |
          RESULT=$(curl -s -X POST "https://$N8N_DOMAIN/rest/owner/setup" \
            -H "Content-Type: application/json" \
            -d "{\"email\":\"$N8N_ADMIN_EMAIL\",\"password\":\"$N8N_OWNER_PASSWORD\",\"firstName\":\"Admin\",\"lastName\":\"Operator\"}")
          echo "$RESULT" | jq '{id: .data.id, email: .data.email}'
          echo "$RESULT" | jq -e '.data.id' > /dev/null || \
            { echo "::warning::Owner setup returned unexpected response — may already be set up"; }

      - name: Store Railway IDs as GitHub Secrets
        env:
          GH_TOKEN: ${{ steps.secrets.outputs.GITHUB_PAT_SECRETS_WRITE }}
          PROJECT_ID: ${{ steps.railway.outputs.project_id }}
          ENV_ID: ${{ steps.railway.outputs.env_id }}
        run: |
          gh secret set RAILWAY_PROJECT_ID --body "$PROJECT_ID" --repo "$GITHUB_REPOSITORY" &
          gh secret set RAILWAY_ENVIRONMENT_ID --body "$ENV_ID" --repo "$GITHUB_REPOSITORY" &
          wait

      - name: Summary
        run: |
          {
            echo "## N8N Deploy Complete"
            echo "| Resource | Status |"
            echo "|---|---|"
            echo "| Railway project \`<project-name>\` | ✅ |"
            echo "| PostgreSQL service | ✅ |"
            echo "| N8N service (\`$N8N_IMAGE\`) | ✅ |"
            echo "| N8N env vars (DB, encryption, auth) | ✅ |"
            echo "| Cloudflare CNAME \`$N8N_DOMAIN\` | ✅ |"
            echo "| N8N healthcheck \`/healthz/readiness → 200\` | ✅ |"
            echo "| N8N owner account \`$N8N_ADMIN_EMAIL\` | ✅ |"
          } >> "$GITHUB_STEP_SUMMARY"
```

## Known Failure Patterns

| Failure | Symptom | Fix |
|---|---|---|
| N8N healthcheck timeout | HTTP 000 after 10 min | Check PostgreSQL connection — verify `DB_POSTGRESDB_HOST` [your-railway] reference var resolved |
| SSL cert not provisioned | TLS error on N8N domain | [your-cloudflare] proxy must be **disabled** (DNS-only) — proxied mode breaks [your-railway]'s ACME challenge |
| Owner setup fails | `{"message":"...already set up"}` | Safe to ignore — N8N was already initialized |
| CNAME points to wrong domain | [your-cloudflare] Error 1016 | Workflow auto-patches CNAME to live [your-railway] domain — re-run if [your-railway] domain changed |
| N8N_ENCRYPTION_KEY rotated | All credentials inside N8N invalid | Never rotate `N8N_ENCRYPTION_KEY` after first deploy — it encrypts stored credentials |
| WIF auth fails | `Permission denied` on GCP SM | Verify `WIF_PROVIDER` and `WIF_SERVICE_ACCOUNT` GitHub Secrets are set |

## Safety Rules

1. **NEVER print, log, or commit any secret value** — all values flow through masked env vars.
2. **NEVER rotate `N8N_ENCRYPTION_KEY`** after first deploy — it breaks all stored N8N credentials.
3. **NEVER overwrite an existing `deploy-n8n.yml`** without printing a warning and waiting for confirmation.
4. **NEVER enable [your-cloudflare] proxy** on the N8N CNAME — DNS-only is required for [your-railway] SSL.
5. **NEVER update `[your-journey-file]` before E2E healthcheck passes** — only document confirmed success.
6. **NEVER proceed** if any prerequisite secret is missing from GCP Secret Manager.
